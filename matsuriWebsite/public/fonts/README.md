# Custom fonts

Put the **More Sugar** font file here so the hero title can use it.

- **File name:** `MoreSugar.otf` or `MoreSugar.ttf`
- **Location:** this folder (`public/fonts/`)

The app will load whichever of the two is present. If your file has a different name, rename it to `MoreSugar.otf` or `MoreSugar.ttf`, or update the `@font-face` paths in `src/index.css`.
